# 📖 Project Overview

Fogg AI is an AI-powered Execution Companion designed to help users overcome procrastination by understanding the psychological barriers that prevent them from taking action.

Unlike traditional productivity applications that rely on reminders and task lists, Fogg AI analyses the user's behavioural state and generates personalised interventions using **Google Gemini AI** and the **Fogg Behaviour Model (B = MAP)**.

Its mission is simple:

> **Help users start, maintain momentum, and complete meaningful work before deadlines are missed.**

---

# 🎯 Problem Statement

Most productivity applications help users organise tasks but fail to solve the real problem—**execution**.

People often procrastinate because of:

- Fear of failure
- Perfectionism
- Task overwhelm
- Planning fallacy
- Low motivation
- Decision fatigue
- Cognitive overload

Fogg AI identifies these psychological barriers and provides personalised AI guidance instead of generic reminders.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/durgesj/Fogg-AI.git
```

Move into the project

```bash
cd Fogg-AI
```

Install dependencies

```bash
npm install
```

Create a local environment file.

```bash
cp .env.example .env.local
```

Configure your environment variables inside `.env.local`.

Start the development server.

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env.local` file and configure the following variables.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY

VITE_FIREBASE_API_KEY=YOUR_FIREBASE_API_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_FIREBASE_AUTH_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_FIREBASE_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET=YOUR_FIREBASE_STORAGE_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_FIREBASE_MESSAGING_SENDER_ID
VITE_FIREBASE_APP_ID=YOUR_FIREBASE_APP_ID
```

> **Important:** Never commit your actual API keys or secrets to GitHub. Use placeholder values in public repositories.

# 🚀 Current Version

The current version is a hackathon-ready prototype.

To ensure a seamless evaluation experience, Fogg AI provides a **Guest Mode** that allows judges and users to explore all AI-powered execution features without authentication.

Google Sign-In has already been integrated using Firebase Authentication. Since the OAuth application is currently in testing mode, Google login is limited to authorised test users. The production version will support secure public Google Sign-In with Cloud Firestore synchronisation.
